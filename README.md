# v3-ts-2021

### 依赖安装
```
npm install
```

### 本地编辑热更新
```
npm run serve
```

### 四种环境发布
```
npm run build_sit // sit环境
npm run build_uat // uat环境
npm run build_pre // pre环境(预发布)
npm run build_prod // prod环境(线上)
```

### 打印项目报告（依赖模块大小报告，性能优化可参考）
```
npm run report
```

### 单元测试（含有代码覆盖率报告）
```
npm run test:unit
```

### 端到端测试
```
npm run test:e2e
```

### Lints and fixes files
```
npm run lint
```
